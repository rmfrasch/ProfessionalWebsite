HARVEST SOFTWARE

Uses Java and JavaFX for GUI. An application that takes creates a field from user parameters, and then prompts the user to input of harvest weight and moistures, normalizes the data to a moisture of 15 percent, puts the resulting output into a 2D array, and puts the unchanged user input into a text file. The user sets the field up by defining parameters.
