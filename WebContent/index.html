HARVEST SOFTWARE

This Harvest Software will record data and associate it with a set of coordinates that relate to a field position. Uses Java and JavaFX for GUI. This application creates a field from user-entered parameters and then prompts the user to input harvest weight and moistures; it will then normalize the data to a moisture of 15 percent, put the resulting output into a 2D array, and output the unchanged user input into a text file. At the end of the program, outputs sorted data from lowest yield to the highest yield into the console using a Binary Tree.

This was a group project, and I was responsible for viewing all appointments, editing appointments, adding Doctors, and viewing doctors. It is meant to be a baseline for a scheduling application in the medical field. Profiles and appointments persist on the mySQL database.
